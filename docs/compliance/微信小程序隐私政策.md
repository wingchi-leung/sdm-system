# SDM 系统微信小程序隐私政策

更新日期：2026年5月18日  
生效日期：2026年5月18日

欢迎使用 SDM 系统微信小程序（以下简称“本小程序”）。我们非常重视您的个人信息与隐私保护。本政策将说明我们如何收集、使用、存储、共享和保护您的个人信息，以及您享有的相关权利。

如您不同意本政策的任何内容，您应立即停止使用本小程序。

## 一、我们如何收集和使用个人信息

我们仅在实现业务功能所必需的范围内收集个人信息。

### 1. 登录与身份识别
- 为实现微信登录，我们会收集微信 `openid`（必要时包括 `session_key` 相关校验信息）。
- 为实现手机号快捷登录（用户主动授权后），我们会收集微信返回的手机号信息。
- 使用目的：账号识别、登录鉴权、会话管理、支付用户绑定。

### 2. 账号资料与活动服务
- 在您完善资料或报名活动时，我们可能收集：姓名、手机号、性别、年龄、职业、行业、邮箱等信息。
- 使用目的：完成账号资料维护、活动报名、活动通知、现场签到管理、活动统计与客户服务。

### 3. 支付服务
- 当您报名付费活动时，我们会处理支付订单相关信息，包括订单号、订单状态、支付金额、支付时间、交易流水号、付款人 `openid` 等必要信息。
- 使用目的：完成微信支付、订单状态查询、支付结果确认、财务对账与风控审计。

### 4. 图片上传（您主动触发）
- 当您主动上传头像、活动海报或社区封面时，我们会访问您选择上传的图片文件。
- 使用目的：展示个人头像、活动内容和社区动态。

### 5. 运营与安全保障
- 为保障系统稳定运行，我们可能记录必要的日志信息（如接口调用时间、错误码、设备环境信息）。
- 我们不会在日志中明文展示完整身份证号、手机号等敏感信息。

## 二、设备权限调用说明

根据业务功能需要，本小程序可能申请以下权限：
- 相册/存储读取权限：用于您主动选择图片并上传头像、海报或封面。
- 相机权限（如启用拍照上传）：用于您主动拍摄并上传图片。

上述权限仅在对应功能场景下，经您主动触发后申请，不会默认强制开启。

## 三、我们如何共享、转让、公开披露个人信息

### 1. 共享
我们不会向无关第三方出售您的个人信息。仅在以下必要场景共享：
- 与微信平台能力共享：用于微信登录、手机号授权、微信支付等能力实现。
- 与受托服务方共享：仅限为实现存储、网络安全、系统运维等必要服务，并要求其履行保密与安全义务。

### 2. 转让
如发生合并、分立、资产转让等情形导致个人信息转移，我们将依法通知您，并要求接收方继续受本政策约束。

### 3. 公开披露
除法律法规要求、行政或司法机关依法要求外，我们不会公开披露您的个人信息。

## 四、我们如何存储和保护个人信息

- 存储地点：中华人民共和国境内服务器。
- 存储期限：在实现业务目的所必需的最短期限内保存；超过期限后将删除或匿名化处理（法律法规另有规定的除外）。
- 安全措施：访问控制、传输安全控制、敏感信息加密存储、日志脱敏、权限最小化管理等。

## 五、您如何管理个人信息

您可通过本小程序内功能行使以下权利：
- 查询与更正：可在“设置/修改个人信息”中查看和更新资料。
- 删除或清理：可申请删除部分个人资料字段（受业务和法律要求限制）。
- 注销账号：可在“设置/注销账号”中发起注销；注销后将停用当前账号登录能力。

如您需进一步行使权利（如复制、更正、删除、撤回同意、投诉建议），可通过本政策“联系我们”章节提供的方式联系。

## 六、未成年人保护

若您是未满14周岁的未成年人，请在监护人陪同下阅读并同意本政策，并在监护人同意后使用本小程序。我们将按照法律法规要求加强未成年人个人信息保护。

## 七、政策更新

我们可能根据业务变化或法律法规要求更新本政策。更新后会通过小程序页面公告、弹窗或其他合理方式提示。若涉及重大变更，我们会以更显著方式通知您。
