# SDM 系统微信小程序用户服务协议

更新日期：2026年5月19日  
生效日期：2026年5月19日

欢迎您使用 SDM 系统微信小程序服务。

本《用户服务协议》（以下简称“本协议”）由您与 SDM 系统微信小程序运营者（以下简称“平台”或“我们”）共同缔结，对双方均具有法律约束力。请您在使用服务前认真阅读并充分理解本协议。

【重要提示】  
请您重点阅读与您权益有重大关系的条款，包括但不限于：账号管理、课程内容使用限制、知识产权、免责条款、违约责任、争议解决等。您勾选同意、点击确认或实际使用服务的，视为您已阅读、理解并同意本协议全部内容。

## 一、服务条款的接受

1. 本协议适用于您通过本小程序使用的全部服务，包括但不限于课程/活动信息浏览、报名、支付、签到、互动等。  
2. 您对本协议的接受方式包括但不限于：勾选同意、点击确认、注册、登录、浏览、报名、支付、发布内容等任一实际使用行为。  
3. 如您不同意本协议任一条款，请立即停止注册或使用本小程序服务。

## 二、定义

1. 用户：指注册、登录、浏览或使用本小程序服务的自然人。  
2. 课程服务：指平台或合作方通过本小程序提供的课程内容、活动服务及其配套服务。  
3. 账号：指用户基于微信授权或手机号认证后获得的用于登录和使用服务的身份凭证。

## 三、用户注册与账号管理

1. 您可通过微信授权登录、手机号授权登录等方式使用服务。  
2. 您应保证提交信息真实、合法、有效，不得冒用他人身份。  
3. 一个手机号仅可注册一个账号，不得通过异常手段重复注册、批量注册或恶意抢注。  
4. 账号仅限账号本人使用，禁止转让、出租、出借、售卖、共享或以其他方式提供他人使用。  
5. 您应妥善保管账号信息及登录状态。因账号保管不善、主动泄露或授权他人使用导致的后果，由您自行承担。  
6. 如发现账号异常、疑似被盗或被冒用，您应立即联系我们。

## 四、课程服务说明

1. 平台提供的服务包括但不限于课程内容展示、活动信息展示、报名候补、订单支付、签到记录、互动内容发布等。  
2. 课程/活动的具体内容、时间安排、费用标准、适用对象、服务边界、退款或改期规则，以页面公示及主办方规则为准。  
3. 平台有权基于产品迭代、运营安排或监管要求对服务内容进行调整、升级或优化，并在合理范围内进行说明。  
4. 因课程安排、服务版本、主办方规则变更产生的影响，平台将在可行范围内通过站内通知、页面公告或其他合理方式告知用户。

## 五、用户行为规范

1. 您在使用服务过程中应遵守中国法律法规、公序良俗及平台规则。  
2. 您不得实施以下行为：  
- 发布违法违规、侵权、侮辱、诽谤、虚假、骚扰性内容；  
- 破坏系统安全、恶意调用接口、抓取数据、干扰平台正常运行；  
- 冒用他人身份、盗用他人账号、未经授权使用他人信息；  
- 未经授权对课程或活动内容进行录音、录像、录屏、翻录、截图整合传播、网盘分发、二次售卖或其他形式倒卖；  
- 从事任何损害平台、其他用户或第三方合法权益的行为。  
3. 对于违规行为，平台有权根据情节采取警告、限制功能、暂停服务、永久封禁账号等处理措施，并保留追究法律责任的权利；涉嫌犯罪的，将依法向有关机关报案并配合调查。

## 六、知识产权声明

1. 本小程序及相关内容（包括但不限于页面设计、程序代码、文字、图片、标识等）的知识产权归平台或合法权利人所有。  
2. 课程内容及相关资料（包括课件、视频、音频、讲义、图文、案例与练习）受著作权法等法律保护。  
3. 未经书面许可，任何单位或个人不得复制、传播、翻录、改编、汇编、反向工程或用于商业化用途。

## 七、免责声明

1. 在法律允许范围内，因下列原因导致服务中断、延迟或异常的，平台不承担由此产生的责任：  
- 系统维护、升级、故障修复；  
- 网络服务故障、通信线路问题、第三方平台故障（如微信平台、支付通道）；  
- 不可抗力或非平台可控因素（如自然灾害、重大公共事件、监管要求）。  
2. 因用户自身原因（包括账号保管不当、误操作、终端设备故障等）造成的损失，由用户自行承担。  
3. 平台将尽合理努力保障服务稳定，并在可行范围内提前公告或及时恢复服务。

2. 因本协议引起或与本协议有关的争议，双方应先友好协商；协商不成的，可向消费者协会等机构申请调解或向主管部门投诉。  
3. 经协商或调解仍未解决的，任一方可向运营者所在地有管辖权的人民法院提起诉讼。

1. 平台有权根据业务发展、监管要求或法律法规变化修订本协议。  
2. 修订后协议将通过小程序内公告、弹窗、站内通知或其他合理方式进行公示。  
3. 修订后的协议自公示或通知载明之日起生效。您在协议更新后继续使用服务的，视为接受更新后的协议；如不同意，应立即停止使用并可申请注销账号。
